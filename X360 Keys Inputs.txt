RB Keys
-------------------------
thanks to this project for pointing me in the right direction for things:
https://github.com/bearzly/RockBandPiano
-------------------------
Face buttons work like a standard Xbox 360 controller

Keys:
C1 is the leftmost key, C3 is the rightmost key

x = this bit doesn't matter
1 = this bit is activated/used by this input
Note name - Input  Decimal value  Binary value

C1  - LT  128  0b 1xxx xxxx
Db1 - LT  64   0b x1xx xxxx
D1  - LT  32   0b xx1x xxxx
Eb1 - LT  16   0b xxx1 xxxx
E1  - LT  8    0b xxxx 1xxx
F1  - LT  4    0b xxxx x1xx
Gb1 - LT  2    0b xxxx xx1x
G1  - LT  1    0b xxxx xxx1

Ab1 - RT  128  0b 1xxx xxxx
A1  - RT  64   0b x1xx xxxx
Bb1 - RT  32   0b xx1x xxxx
B1  - RT  16   0b xxx1 xxxx
C2  - RT  8    0b xxxx 1xxx
Db2 - RT  4    0b xxxx x1xx
D2  - RT  2    0b xxxx xx1x
Eb2 - RT  1    0b xxxx xxx1

E2  - L Stick X  128  0b xxxx xxxx 1xxx xxxx
F2  - L Stick X  64   0b xxxx xxxx x1xx xxxx
Gb2 - L Stick X  32   0b xxxx xxxx xx1x xxxx
G2  - L Stick X  16   0b xxxx xxxx xxx1 xxxx
Ab2 - L Stick X  8    0b xxxx xxxx xxxx 1xxx
A2  - L Stick X  4    0b xxxx xxxx xxxx x1xx
Bb2 - L Stick X  2    0b xxxx xxxx xxxx xx1x
B2  - L Stick X  1    0b xxxx xxxx xxxx xxx1

C3  - L Stick X  -32768  0b 1xxx xxxx xxxx xxxx

Velocity:
	1st key: L Stick X  0b x111 1111 xxxx xxxx
	2nd key: L Stick Y  0b xxxx xxxx x111 1111
	3rd key: L Stick Y  0b x111 1111 xxxx xxxx
	4th key: R Stick X  0b xxxx xxxx x111 1111
	5th key: R Stick X  0b x111 1111 xxxx xxxx
	further keys don't register velocities

Overdrive button: Right Stick Y  128  0b xxxx xxxx 1xxx xxxx
Pedal port: Right Stick Y  -32768  0b 1xxx xxxx xxxx xxxx

Right Stick Y 32512 0b x111 1111 xxxx xxxx is always active for some reason

Effect slider: just increments the XInput packet count as far as I can tell, no inputs assigned to it. Probably not possible to get its data via XInput, but it's gotta be possible some way else, RB3 can get position data from it.
