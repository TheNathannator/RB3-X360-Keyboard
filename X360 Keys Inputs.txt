RB Keys
-------------------------
thanks to this project for pointing me in the right direction for some things:
https://github.com/bearzly/RockBandPiano

values collected using AntiMicro:
https://github.com/AntiMicro/antimicro
and converted from decimal to binary via the Windows calculator in Programmer mode
-------------------------

Face buttons work like a standard Xbox 360 controller

Keys:
C1 is the leftmost key, C3 is the rightmost key
AntiMicro reports the trigger values as a 16 bit integer, but the implementation that RockBandPiano uses reports the trigger values as a byte. These contain the value reported by AntiMicro, converted to binary. I'll try to collect data directly from XInput at some point.

x = this bit doesn't matter
1 = this bit is activated/used by this input

Velocity:
	1st key: L Stick X  0b x111 1111 xxxx xxxx
	2nd key: L Stick Y  0b xxxx xxxx x111 1111
	3rd key: L Stick Y  0b x111 1111 xxxx xxxx
	4th key: R Stick X  0b xxxx xxxx x111 1111
	5th key: R Stick X  0b x111 1111 xxxx xxxx
	further keys don't register velocities

Overdrive button: Right Stick Y  128  0b xxxx xxxx 1xxx xxxx
	also, 0b xxxx xxxx x000 0000 when first powered on, then 0b xxxx xxxx x111 1111 (127) after it's pressed for the first time 

Effect slider: just increments the XInput packet count, no inputs assigned to it
